---

## 1. 회사 한 줄 소개

**팀의 아이디어를 실행으로 연결하는 실시간 협업 플랫폼, 콜라보**

---

## 2. 시장 기회

### 글로벌 협업 도구 시장

- **시장 규모**: 글로벌 협업 소프트웨어 시장 $20B+ (2025)
- **성장률**: 연평균 13% 성장 전망
- **트렌드**: 원격/하이브리드 근무 확산 → 실시간 협업 도구 수요 급증

### 문제 정의

| 기존 도구 | 한계 |
|-----------|------|
| 화이트보드 도구 (Miro 등) | 아이디어 → 실행 연결 불가. 브레인스토밍에서 끝남 |
| PM 도구 (Monday, Asana 등) | 실시간 협업/비주얼 사고 불가. 태스크 관리만 가능 |
| 올인원 (Notion 등) | 실시간 동시 편집에 약함. 문서 중심 |

**콜라보의 포지션**: 화이트보드 + 프로젝트 보드를 하나의 워크스페이스에서 실시간으로 연결

## 5. 경쟁 우위

### 왜 콜라보인가?

1. **통합 경험**: 화이트보드 ↔ 프로젝트보드가 실시간 연결 (경쟁사는 별도 도구)
2. **속도**: CRDT 기반 실시간 동기화 — 100명 동시 편집에도 지연 없음
3. **전환 비용**: 화이트보드 + PM을 하나로 쓰면 다른 도구로 돌아가기 어려움 (높은 Switching Cost)
4. **AI 선점**: "Cody"로 브레인스토밍 → 태스크 변환 → 진행 관리를 AI가 보조
